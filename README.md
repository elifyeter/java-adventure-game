# Adventure Game

## About
A text-based adventure game written in Java.  
This project was created while following an online course. It demonstrates basic **Object-Oriented Programming (OOP)** concepts like classes, objects, inheritance, and encapsulation.

## Features
- Explore different locations
- Fight monsters
- Manage inventory
- Simple text-based interface

## How to Run
1. Clone the repository
2. Open in Eclipse or your favorite IDE
3. Run `Main.java` from src folder
